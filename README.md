## ğŸš€ Getting Started

### Prerequisites
- Modern web browser
- Text editor (VS Code recommended)
- Git (for version control)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/luke8089/portfolio.git
```

2. Navigate to the project directory:
```bash
cd portfolio
```

3. Open in your browser:
```bash
# Simply open index.html in your browser
# Or use a local server (recommended):
npx serve .
```

### Configuration

1. **Google Analytics**: Replace `GA_MEASUREMENT_ID` in index.html with your GA4 ID
2. **Google Search Console**: Add your verification code in the meta tag
3. **Contact Form**: Update Web3Forms access key if needed
4. **Personal Information**: Update all personal details, links, and content
